# General Instructions
* keep cognitive complexity to 15
* use tailwindcss for styling
* use motion for animations
* use shadcn/ui for UI components
* use tanstack/react-query for data fetching
* use tanstack/react-router for routing
* use @tanstack/react-virtual for virtualized lists
* use @tanstack/react-table for headles tables management
* use @tanstack/react-forms for forms
* use @tanstack/react-query-devtools for debugging queries
* front facing strings should be in a separate file
* logic and UI should be separated
* Mark the props of the component as read-only 
* Prefer using nullish coalescing operator (`??`) instead of a logical or (`||`), as it is a safer operator.
